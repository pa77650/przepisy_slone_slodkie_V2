🥗 Generator przepisów „z tego co mam w lodówce”

💡 Opis projektu
Aplikacja wykorzystująca sztuczną inteligencję do generowania przepisów kulinarnych na podstawie dostępnych składników — zarówno wpisanych ręcznie, jak i rozpoznanych ze zdjęcia.

⚙️ Jak działa aplikacja?

📥 Wejście:
lista składników wpisana przez użytkownika

📸 zdjęcie produktów (jpg/png)

🤖 AI processing:
model image-to-text rozpoznaje składniki ze zdjęcia
przekształcenie danych nieustrukturyzowanych → w uporządkowaną listę produktów
analiza składników i dopasowanie przepisów

📤 Wyjście:
gotowe przepisy dopasowane do dostępnych produktów
sugestie dań minimalizujące marnowanie jedzenia

🚀 Kluczowe funkcjonalności
✅ Generowanie przepisów na podstawie składników
✅ Rozpoznawanie produktów ze zdjęć (Computer Vision)
✅ Automatyczne przetwarzanie danych (image → text → recipe)
✅ Wsparcie planowania posiłków
✅ Redukcja food waste

🧠 Technologie i koncepcje
🤖 AI / Machine Learning
👁️ Computer Vision (image-to-text)
💬 NLP (generowanie tekstu)
🔄 Data transformation (unstructured → structured)
